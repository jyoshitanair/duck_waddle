DESCRIPTION
---
You are a duck just trying to have a swim. But oh no...there are so many rocks in your path...


try and dodge them or you will die! 



HOW TO PLAY
---
⭐️You can move side to side using the arrow keys (left and right) 


⭐️Jump with the up arrow (but you can not jump over rocks)


⭐️Enter a new name each time you play :3


⭐️When you die click on the try again to return to the start or the leaderboard to see the leaderboard


⭐️have fun!



FEATURES
---
⭐️ Signal Buses: Manager, SilentWolf, MusicManager


⭐️ Backend for leaderboard using SilentWolf


⭐️3D models, cool fonts, and models!


⭐️Dynamic loading of the water and logs to save memory (deleted after about 1.5 roads have passed)


⭐️Water Shader 


⭐️Blender Meshes!
